Panorama
========

The Panorama property holds the panorama created from the point cloud, depending on a property
(:class:`ColorProperty.ColorProperty`, :class:`CurvatureProperty.CurvatureProperty`, etc.).
These are created based on :class:`SphericalCoordinates.SphericalCoordinates` class).
