class PointSet.PointSet(points, **kwargs)[source]¶

Basic point cloud

Mandatory Data (must be different from None):

__xyz (nX3 ndarray, n-number of points) - xyz unstructured Data (only 3D currently)
Optional Data (May not exist at all, or can be None):
__rgb - color of each point (ndarray) __intensity - intensity of each point (ndarray) __measurement_accuracy: noise of modeled surface
