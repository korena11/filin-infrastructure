setValues(*args, **kwargs)[source]¶

Sets the values into the panoramaProperty object

Parameters:
  • panoramaData – The data to be represented as a panorama (e.g. range, intesity, etc.). Default: range
  • rowIndexes – The row indices of the points in the point set based on the elevation angles
  • columnIndexs – The column indices of the points in the point set based on the azimuth angles

Optionals

Parameters:
  • dataType (str) – A string indicating the type of data stored in the panoramic view (e.g. ‘range’, ‘intensity’, etc.)
  • minAzimuth (float) – The minimal azimuth value
  • maxAzimuth (float) – The maximal azimuth value
  • minElevation (float) – The minimal elevation value
  • maxElevation (float) – The maximal elevation value
  • azimuthSpacing (float) – The measurements’ angular resolution in the azimuth direction.
  • elevationSpacing (float) – The measurements’ angular resolution in the elevation angle direction

Note

For the Scanstation C10 the measurements’ angular resolution for both elevation and azimuth directions:

  • Low: 0.11 deg
  • Medium: 0.057 deg
  • High: 0.028 deg
