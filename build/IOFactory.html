Reading points from .pts file. If the pts file holds more than one PointSet merge into one PointSet (unless told

otherwise).

Parameters:
  • fileName – name of .pts file
  • pointsetlist – placeholder for created PointSet objects
  • colorslist – placeholder for ColorProperty for PointSet object(s)

Optionals :param merge: merge points in file into one PointSet or not. Default: True.

Returns: The created PointSet or the list of the PointSets created and the ColorProperty that belongs to it
Return type: tuple
